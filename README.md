# Langchain Express 

Explore Neo4J RAG LLM pipeline with custom chaings and agents for Hybrid Cloud Mesh and other purposes

## Development server

Run `npm run watch:deploy` for a dev server. The application will automatically reload if you change any of the source files.  

To interact with Langchain Express Server, head over to `https://github.ibm.com/ljeff/mesh-genai` and clone the repo.  Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`.  Navigate to `Langchain JS` tab, select settings and provide Langchain Express URL.

## Vector Storage

`docker compose up -d` to spin up local ChromaDB
